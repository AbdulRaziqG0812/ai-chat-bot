# AI Chat Bot 🤖

An AI-powered chatbot built using **Python Flask** and the **Groq API**. The chatbot provides real-time responses through a clean and user-friendly web interface.

## Features

- AI-powered chatbot
- Real-time responses
- Simple and responsive UI
- Flask backend
- Groq API integration
- Chat history support

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Groq API

## Installation

Clone the repository:

```bash
git clone https://github.com/AbdulRaziqG0812/ai-chat-bot.git
```

Go to the project folder:

```bash
cd ai-chat-bot
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Environment Variable

Create an environment variable for your Groq API Key.

Mac/Linux

```bash
export GROQ_API_KEY="YOUR_GROQ_API_KEY"
```

Windows (CMD)

```cmd
set GROQ_API_KEY=YOUR_GROQ_API_KEY
```

Windows (PowerShell)

```powershell
$env:GROQ_API_KEY="YOUR_GROQ_API_KEY"
```

## Run the Project

```bash
python app.py
```

The application will start on:

```
http://127.0.0.1:5000
```
